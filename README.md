# Medical Insurance Dataset – Data Cleaning Project
## Project Overview
- This project focuses on cleaning and preprocessing a medical insurance dataset using "Python (Pandas)".

###The raw dataset contained inconsistencies such as:
- Duplicate records
- Missing values
- Inconsistent column formatting
- Unstandardized categorical values

###The cleaned dataset is ready for:
- Data analysis
- Visualization
- Machine learning modeling

## Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook
